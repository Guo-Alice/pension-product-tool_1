# 个人养老金产品搜索与决策工具

## 项目简介

基于中国个人养老金政策的智能产品推荐系统，帮助用户根据个人情况选择适合的养老金产品。系统通过分析用户画像和产品特征，提供个性化的产品推荐、对比分析和决策支持。

## 功能特点

### 1. 智能推荐
- 基于多维度用户画像（年龄、收入、风险偏好等）推荐产品
- 个性化匹配算法，考虑年龄、收入、风险、社保等多种因素
- 动态权重调整，优化推荐结果

### 2. 产品对比
- 多产品并行对比分析
- 核心参数对比（保费、保障、风险等）
- 可视化对比报告生成

### 3. 数据分析
- 产品类型分布统计
- 风险等级分析
- 保费区间统计
- 交互式图表展示

### 4. 用户管理
- 用户画像保存与加载
- 推荐历史记录
- 个性化建议生成

### 5. 数据管理
- Excel数据导入与处理
- 处理结果导出（CSV、Excel、JSON）
- 数据备份与恢复
